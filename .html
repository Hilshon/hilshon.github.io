Классификация инструментов

Mind Map с различными инструментами для создания цифрового контента.

Инструмент 1: Canva

Canva - удобный онлайн-сервис для создания графики и презентаций.

Примеры: баннеры, посты для соцсетей, инфографика.

Инструмент 2: Figma

Figma - мощный инструмент для дизайна интерфейсов и прототипирования.

Примеры: веб-дизайн, UI/UX-дизайн, прототипы.

Инструмент 3: Tilda

Tilda - платформа для создания лендингов и сайтов без программирования.

Примеры: лендинги, блоги, онлайн-магазины.
